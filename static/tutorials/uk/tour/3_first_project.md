### Код

Код для твого проєкта готовий і вже містить три визначені функції:
```init```, ```update``` й ```draw```. Ми попрацюємо над вмістом функції
```draw```. Додай наступний рядок між рядками
```draw = function()``` ось тут ```end```:

```
  screen.drawSprite("sprite",0,0,20)
```

Ось як тепер виглядатиме твій код.
```
draw = function()
  screen.drawSprite("sprite",0,0,20)
end
```

### Виконання

Тепер твій персонаж відображається по центру вікна виконання. Рядок коду який ми додали
викликає функцію ```drawSprite``` в об'єкті ```screen```. Виклик здійснюється з такими параметрами:
імʼя спрайта який потрібно відобразити ```"sprite"``` (переконайся, що це справді ім'я спрайта, який ти створив),
x та y координати місця для відображення (0,0 це центр екрану) і розмір відображення (20).

Ти можеш погратися з координатами, що змінить позицію відображення твого спрайта.
Ти можеш внести власні зміни і одразу побачити їх в реальному часі у вікні виконання. 

## Додавання фону

:navigate projects.code

### Додавання фонового кольору

Над нашим рядком ```screen.drawSprite(...)``` ми додамо наступний рядок:
```
  screen.fillRect(0,0,400,400,"#468")
```

```fillRect``` значить "fill a rectangle — заповнити прямокутник". Параметр ```"#468"```
означає синьо-сірий колір. Натисни на це після чого натисний й утримуй кнопку CTRL,
з’явиться засіб для вибору кольору.

### Керування персонажем

Для керування позицією малювання персонажа ми використовуємо дві змінні: ```x``` та ```y```.
Давай змінимо рядок коду, який малює спрайт наступним чином:

```
  screen.drawSprite("sprite",x,y,20)
```

Персонаж тепер малюється на координатах ```x``` , ```y```.

## Керування

:navigate projects.code

### Керування персонажем

Все що нам потрібно зараз - це змінювати значення ```x``` та ```y``` коли
натискаються відповідні клавіші зі стрілками на клавіатурі. Встав наступний рядок
між ```update = function()``` і ```end```:

```
  if keyboard.LEFT then x = x-1 end
```

Твій повний код тепер виглядає наступним чином:

```
init = function()
end

update = function()
  if keyboard.LEFT then x = x-1 end
end

draw = function()
  screen.fillRect(0,0,400,400,"rgb(140,198,110)")
  screen.drawSprite("sprite",x,y,20)
end
```

## Керування персонажем

:navigate projects.code

### Керування персонажем

Натисни на вікно виконання, тепер натискани на клавіатурі стрілку вліво.
Ти побачиш як твій персонаж буде рухатися в ліву сторону.

Чому це відбувається: ми додали рядок коду який перевіряє, чи натиснута стрілка вліво (```keyboard.LEFT```)
і коли це відбувається, значення ```x``` зменшується на 1.

Знаючи що інші клавіші позначаються як RIGHT, UP і DOWN, додамо три рядки до твого коду, забезпечуючи
твого персонажа можливістю переміщуватися у будь-якому напрямку.

(Рішення на наступному кроці)

## Керування персонажем

:navigate projects.code

### Керування персонажем

Це повний код функції ```update``` який дозволяє переміщувати персонажа у всі 4 сторони натискаючи
на клавіатурі кнопки з відповідними напрямками стрілок.

```
update = function()
  if keyboard.LEFT then x = x-1 end
  if keyboard.RIGHT then x = x+1 end
  if keyboard.UP then y = y+1 end
  if keyboard.DOWN then y = y-1 end
end
```
