**microStudio** — безкоштовне середовище для розробки ігор. Воно також прагне стати платформою для навчання та обміну досвідом.

**microStudio** - проєкт з віткритим вихідним кодом, розповсюджується під ліцензією MIT: https://github.com/pmgl/microstudio/
